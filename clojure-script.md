# Clojure Script

ClojureScript는 클라이언트에서 로직을 작성하기위해 자바스크립트 대신 사용할 수 있는 훌륭한 대안이다. ClojureScript 를 사용하는 장점은 다음과 같다:

* 클라이언트와 서버에서 같은 언어로 작성이 가능
* 프론트엔드와 백엔드가 같은 코드 공유 가능
* 깔끔하고 일관된 언어
* Leiningen 으로하는 의존성 관리
* 불변\(immutable\) 데이터 구조
* 강력한 표준 라이브러리
